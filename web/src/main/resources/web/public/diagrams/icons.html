Manipulación vectorial

En caso de que se desee acotar el número y tipo de componentes que puede el editor utilizar en cada tipo de diagrama, pueden diseñarse componentes gráficos en formato SVG, que ofrece la flexibilidad de escalado prácticamente infinito sin pérdida de calidad. Esto favorecerá el estrablecimiento de zooming como estrategia de visualización de muchos componentes en una única visual
