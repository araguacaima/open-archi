Otro modo de visualizar secuencias y flujos

Este ejemplo muestra un monitoreo contínuo del estado de un número de componentes. Cada dos segundos actualiza la pantalla, mostrando algunos problemas aleatorios resaltando el componente donde se halló la falla. Si lograramos incorporar información de reliability podríamos representar posibles lack, cuellos de botella y casos bordes en tiempo de diseño.
