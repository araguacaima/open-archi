Diagrama de Gantt para representar Release Planning

Cualquier componente tiene sub-componentes o partes. También tienes features que supondrán ciertas tareas a concretarse para poder disponibilizarse. En OpenArchi queremos tener una visual integrada de todos los pormenores que hacen que un determinado componente puede conformar parte de un modelo de solución. Es posible extraer el release plan definido para cada característica que se ofrezca y visualizarlo en tiempo real en cada componente.
