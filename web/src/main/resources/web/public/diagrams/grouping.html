Alternativa para la expansión de componentes

Esto es sólo una forma de representar componentes que tienen otros componentes dentro de sí. La idea de fondo es poder establecer vistas que varien en profundidas, dependiendo del consumidor. A álto nivel estarían las piezas primarias de cualquier modelo. Dependiendo de la interacción del usuario será posible, sin salir de la pantalla de visualización, expandir la visual, incorporando los elementos internos de cada componente, para entender mejor su funcionamiento y al forma real como se interrelaciona con el resto
